Méthode setTime (ISQLServerPreparedStatement)

Définit le paramètre désigné selon la valeur d'heure spécifiée.

Depuis la version 3.0 du pilote JDBC SQL Server, le comportement de cette méthode est modifié par la propriété de connexion sendTimeAsDatetime (Définition des propriétés de connexion) et SQLServerDataSource.setSendTimeAsDatetime.

Pour plus d'informations, consultez Configuration du mode d'envoi des valeurs java.sql.Time au serveur.

Liste de surcharge

Nom

Description

setTime (int, java.sql.Time)

Définit le paramètre désigné selon la valeur d'heure spécifiée.

setTime (int, java.sql.Time, java.util.Calendar)

Définit le paramètre désigné selon les valeurs d'heure et de calendrier spécifiées.
