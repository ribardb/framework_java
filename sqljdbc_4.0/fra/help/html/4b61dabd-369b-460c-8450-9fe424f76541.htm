Description

addConnectionEventListener

(Hérité de SQLServerPooledConnection.) Inscrit l'écouteur d'événement donné afin qu'il puisse être informé lorsqu'un événement se produit sur cet objet Connection.

(Hérité de SQLServerPooledConnection.) Ferme la connexion physique représentée par cet objet Connection.

(Hérité de SQLServerPooledConnection.) Crée un descripteur d'objet pour la connexion physique représentée par cet objet Connection.

getXAResource

Récupère un objet SQLServerXAResource que le gestionnaire de transactions utilisera pour gérer la participation de cet objet SQLServerXAConnection dans une transaction distribuée.

(Hérité de SQLServerPooledConnection.) Supprime l'écouteur d'événement donné.
