Méthode updateNClob (int, java.io.Reader)

Met à jour la colonne désignée à l'aide de l'objet Reader spécifié.

Syntaxe

public void updateNClob(int columnIndex,
                        java.io.Reader reader)

Paramètres

columnIndex

int qui indique l'index de colonne.

reader

Objet Reader.

Exceptions

SQLServerException

Notes

Cette méthode updateNClob est spécifiée par la méthode updateNClob dans l'interface java.sql.ResultSet.

Cette méthode est uniquement prise en charge sur les colonnes nvarchar(max), ntext et xml. L'utilisation de cette méthode sur n'importe quel autre type de données entraîne la levée d'une exception.
