Utilisation des instructions avec SQL

Lorsque vous utilisez les données d'une base de données SQL Server en utilisant le Pilote JDBC Microsoft pour SQL Server et les instructions SQL incluses, vous pouvez utiliser différentes classes. La classe que vous utilisez dépend du type d'instruction SQL que vous souhaitez exécuter.

Si votre instruction SQL ne contient pas de paramètres IN, utilisez la classe SQLServerStatement. Dans le cas contraire, utilisez la classe SQLServerPreparedStatement.

Si vous devez utiliser des instructions SQL contenant à la fois des paramètres IN et OUT, vous devez les implémenter en tant que procédures stockées et les appeler à l'aide de la classe SQLServerCallableStatement. Pour plus d'informations sur l'utilisation des procédures stockées, consultez Utilisation d'instructions avec des procédures stockées.
