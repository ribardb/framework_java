Méthode isSameRM (SQLServerXAResource)

Détermine si l'instance de gestionnaire de ressources représentée par l'objet cible est identique à l'instance représentée par l'objet XAResource donné.

Syntaxe

public boolean isSameRM(javax.transaction.xa.XAResource xares)

Paramètres

xares

Objet XAResource.

Valeur de retour

true si les instances sont identiques. Sinon, false.

Exceptions

javax.transaction.xa.XAException

Notes

Cette méthode commit est spécifiée par la méthode commit dans l'interface javax.transaction.xa.XAResource.
