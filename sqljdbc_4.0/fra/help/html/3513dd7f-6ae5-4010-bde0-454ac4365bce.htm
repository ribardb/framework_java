Méthode getUser (SQLServerDataSource)

Retourne le nom d'utilisateur qui est utilisé pour la connexion à la source de données.

Syntaxe

public java.lang.String getUser()

Valeur de retour

String qui contient le nom d'utilisateur.

Notes

La méthode setUser définit le nom d'utilisateur qui sera utilisé lors de la connexion à l'instance de SQL Server. Si la valeur de nom d'utilisateur n'est pas définie, la méthode getUser retourne la valeur par défaut Null.
