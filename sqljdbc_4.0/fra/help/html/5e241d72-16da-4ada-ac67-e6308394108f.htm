Obtention de la version du pilote

La version installée du Pilote JDBC Microsoft pour SQL Server peut être déterminée comme suit :

  • Appelez les méthodes SQLServerDatabaseMetaDatagetDriverMajorVersion, getDriverMinorVersion ou getDriverVersion.

  • La version est affichée dans le fichier lisezmoi.txt de la distribution de produit.

De plus, le nom du pilote JDBC peut être retourné à partir de l'appel de la méthode getDriverName sur la classe SQLServerDatabaseMetaData. Cet appel retourne « Microsoft JDBC Driver 4.0 pour SQL Server ».

Voici un exemple de sortie des appels aux méthodes de la classe SQLServerDatabaseMetaData :

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

Où « xxx.x » est le numéro de version finale.
