Méthode setNull (java.lang.String, int, java.lang.String)

Définit le paramètre désigné avec une valeur Null, en fonction du type et du nom du paramètre à définir.

Syntaxe

public void setNull(java.lang.String sCol,
                    int nType,
                    java.lang.String sTypeName)

Paramètres

sCol

String qui contient le nom du paramètre.

nType

Code de type JDBC qui est défini par java.sql.Types.

sTypeName

String qui indique le nom complet du paramètre en cours de définition.

Exceptions

SQLServerException

Notes

Cette méthode setNull est spécifiée par la méthode setNull dans l'interface java.sql.CallableStatement.
