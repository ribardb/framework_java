Méthode executeUpdate (java.lang.String, int[])

Exécute l'instruction SQL fournie et signale au Pilote JDBC Microsoft pour SQL Server que les clés générées automatiquement qui sont indiquées dans le tableau spécifié doivent être disponibles pour la récupération.

Syntaxe

public final int executeUpdate(java.lang.String sql,
                               int[] columnIndexes)

Paramètres

sql

String qui contient une instruction SQL.

columnIndexes

Tableau de int qui indique les index de colonne des clés générées automatiquement qui doivent être disponibles.

Valeur de retour

int qui indique le nombre de lignes affectées ou 0 en cas d'utilisation d'une instruction DDL.

Exceptions

SQLServerException

Notes

Cette méthode executeUpdate est spécifiée par la méthode executeUpdate dans l'interface java.sql.Statement.

Si vous exécutez les résultats d'une procédure stockée dans plusieurs mises à jour, ou dans une mise à jour qui retourne plus d'un jeu de résultats, vous pouvez utiliser la méthode execute pour exécuter la procédure stockée.
