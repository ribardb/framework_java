Méthode getClob (int) (ISQLServerResultSet)

Récupère la valeur de l'index de colonne désigné dans la ligne actuelle de cet objet SQLServerResultSet en tant qu'objet Clob dans le langage de programmation Java.

Syntaxe

public java.sql.Clob getClob(int columnIndex)

Paramètres

columnIndex

int qui indique l'index de colonne.

Valeur de retour

Objet Clob.

Exceptions

SQLServerException

Notes

Cette méthode getClob est spécifiée par la méthode getClob dans l'interface java.sql.ResultSet.
