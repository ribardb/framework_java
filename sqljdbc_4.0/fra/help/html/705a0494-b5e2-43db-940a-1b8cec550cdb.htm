Méthode setSendTimeAsDatetime (SQLServerDataSource)

Cette méthode a été ajoutée dans la version 3.0 du pilote JDBC SQL Server.

Modifie le paramètre de la propriété de connexion sendTimeAsDatetime.

Syntaxe

public void setSendTimeAsDatetime(boolean sendTimeAsDateTime)

Paramètres

sendTimeAsDateTime

Valeur booléenne. Lorsque la valeur True est définie, les valeurs java.sql.Time sont envoyées au serveur en tant que types SQL Serverdatetime. Lorsque la valeur False est définie, les valeurs java.sql.Time sont envoyées au serveur en tant que types SQL Servertime.

Notes

SQLServerDataSource.getSendTimeAsDatetime retourne le paramètre de la propriété de connexion sendTimeAsDatetime.

Pour plus d'informations sur la propriété de connexion sendTimeAsDatetime, consultez Définition des propriétés de connexion.

Pour plus d'informations, consultez Configuration du mode d'envoi des valeurs java.sql.Time au serveur.
