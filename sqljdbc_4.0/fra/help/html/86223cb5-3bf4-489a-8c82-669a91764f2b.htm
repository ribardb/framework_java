Méthode getMetaData (ISQLServerConnection)

Récupère un objet SQLServerDatabaseMetaData qui contient des métadonnées sur la base de données pour lequel cet objet SQLServerConnection représente une connexion.

Syntaxe

public java.sql.DatabaseMetaData getMetaData()

Valeur de retour

Objet DatabaseMetaData.

Exceptions

SQLServerException

Notes

Cette méthode getMetaData est spécifiée par la méthode getMetaData dans l'interface java.sql.Connection.
