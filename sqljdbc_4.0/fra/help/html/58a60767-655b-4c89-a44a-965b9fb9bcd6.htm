Méthode getFloat (SQLServerCallableStatement)

Récupère la valeur du paramètre désigné en tant qu'objet float dans le langage de programmation Java.

Liste de surcharge

Nom

Description

getFloat (int)

Récupère la valeur du paramètre désigné en tant qu'objet float dans le langage de programmation Java selon l'index de paramètre.

getFloat (java.lang.String)

Récupère la valeur du paramètre désigné en tant qu'objet float dans le langage de programmation Java selon le nom du paramètre.
