Connexion à SQL Server à l'aide du pilote JDBC

L'une des principales choses à faire avec le Pilote JDBC Microsoft pour SQL Server est d'établir une connexion à une base de données SQL Server. Toute interaction avec la base de données se produit via l'objet SQLServerConnection et, comme le pilote JDBC présente une architecture très plate, pratiquement tout comportement intéressant touche l'objet SQLServerConnection.

Si un serveur SQL Server écoute uniquement sur le port IPv6, définissez la propriété système java.net.preferIPv6Addresses pour vous assurer que IPv6 est utilisé à la place de IPv4 pour la connexion à SQL Server :

System.setProperty("java.net.preferIPv6Addresses", "true");

Les rubriques de cette section décrivent la création et le fonctionnement d'une connexion à une base de données SQL Server.
