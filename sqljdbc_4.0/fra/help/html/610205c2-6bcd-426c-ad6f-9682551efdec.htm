Méthode executeQuery (java.lang.String)

Exécute l'instruction SQL donnée et retourne un seul objet SQLServerResultSet.

Syntaxe

public final java.sql.ResultSet executeQuery(java.lang.String sql)

Paramètres

sql

String qui contient une instruction SQL.

Valeur de retour

Objet SQLServerResultSet.

Exceptions

SQLServerException

Notes

Cette méthode executeQuery est spécifiée par la méthode executeQuery dans l'interface java.sql.Statement.

Cette méthode remplace la méthode executeQuery qui se trouve dans la classe SQLServerStatement.

L'appel de cette méthode entraîne une exception étant donné que l'instruction SQL pour l'objet SQLServerPreparedStatement est spécifiée lors de la création de l'objet.

Une exception SQLServerException est levée si l'instruction SQL donnée produit tout autre élément qu'un objet SQLServerResultSet unique.
