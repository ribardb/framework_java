Méthode getSchemaName (SQLServerResultSetMetaData)

Obtient le nom de schéma de la table pour la colonne désignée.

Syntaxe

public java.lang.String getSchemaName(int column)

Paramètres

column

int qui indique l'index de colonne.

Valeur de retour

String qui contient le nom du schéma.

Exceptions

SQLServerException

Notes

Cette méthode getSchemaName est spécifiée par la méthode getSchemaName dans l'interface java.sql.ResultSetMetaData.
