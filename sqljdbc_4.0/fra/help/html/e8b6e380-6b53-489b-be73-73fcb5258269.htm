setDateTimeOffset(int, java.sql.DateTimeOffset) (SQLServerStatement)

Définit le paramètre désigné selon la valeur DateTimeOffset donnée.

Syntaxe

public void setDateTimeOffset(int parameterIndex, DateTimeOffset dateTime)

Paramètres

parameterIndex

Index de la colonne à définir.

dateTimeOffset

Objet DateTimeOffset.

Exceptions

SQLServerException

Notes

Le format DateTimeOffset est « AAAA-MM-JJ HH-MM-SS[.nnnnnnn] [+][-] HH:MM ». Utilisez le tableau suivant à titre de référence.

Type SQL

Insert

datetime

Peut insérer uniquement : « AAAA-MM-JJ hh:mm:ss[.nnn] »

smalldatetime

Peut insérer uniquement : « AAAA-MM-JJ hh:mm:ss »

Time

Peut insérer uniquement : « hh:mm:ss[.nnnnnnn] »

Date

Peut insérer uniquement : « AAAA-MM-JJ »

DateTime2

Peut insérer uniquement : « AAAA-MM-JJ hh:mm:ss[.nnnnnnn] »
