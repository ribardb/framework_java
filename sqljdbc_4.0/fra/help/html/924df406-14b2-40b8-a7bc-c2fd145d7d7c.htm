Méthode getObject (ISQLServerResultSet)

Obtient la valeur de la colonne désignée dans la ligne actuelle de cet objet SQLServerResultSet en tant qu'objet dans le langage de programmation Java.

Liste de surcharge

Nom

Description

Obtient la valeur de l'index de colonne désigné dans la ligne actuelle de cet objet SQLServerResultSet en tant qu'objet dans le langage de programmation Java.

Obtient la valeur de l'index de colonne désigné dans la ligne actuelle de cet objet SQLServerResultSet en tant qu'objet dans le langage de programmation Java, en fonction de l'objet Map donné.

getObject (java.lang.String)
