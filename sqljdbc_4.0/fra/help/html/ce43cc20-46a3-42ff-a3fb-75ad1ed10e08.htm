Connexion et extraction de données

Lorsque vous travaillez avec le Pilote JDBC Microsoft pour SQL Server, il existe deux méthodes principales pour établir une connexion à une base de données SQL Server. La première méthode consiste à définir les propriétés de connexion de l'URL de connexion, puis à appeler la méthode getConnection de la classe DriverManager pour retourner un objet SQLServerConnection.

Pour obtenir une liste des propriétés de connexion prises en charge par le pilote JDBC, consultez Définition des propriétés de connexion.

La seconde méthode implique de définir les propriétés de connexion grâce aux méthodes setter de la classe SQLServerDataSource, puis d'appeler la méthode getConnection pour retourner un objet SQLServerConnection.

Les rubriques de cette section décrivent les différentes façons de se connecter à une base de données SQL Server ; elles présentent également différentes techniques d'extraction de données.
