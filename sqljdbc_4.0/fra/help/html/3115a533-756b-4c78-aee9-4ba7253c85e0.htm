Méthodes

Nom

Description

clearWarnings

Efface tous les avertissements signalés pour cet objet SQLServerConnection.

close

Libère immédiatement les ressources de base de données et JDBC pour cet objet SQLServerConnection, au lieu de patienter jusqu'à leur libération automatique.

validation

Rend permanentes toutes les modifications effectuées depuis la précédente validation ou restauration, et libère tous les verrous de base de données actuellement maintenus par cet objet SQLServerConnection.

createBlob

Crée un objet java.sql.Blob sans aucune donnée.

createClob

Crée un objet java.sql.Clob sans aucune donnée.

createNClob

Crée un objet java.sql.NClob sans aucune donnée.

createStatement

Crée un objet SQLServerStatement pour l'envoi d'instructions SQL à la base de données.

createSQLXML

Crée un objet java.sql.SQLXML sans aucune donnée.

getAutoCommit

Récupère le mode de validation automatique actuel pour cet objet SQLServerConnection.

getCatalog

Récupère le nom de catalogue actuel pour cet objet SQLServerConnection.

getClientConnectionID, méthode (SQLServerConnection)

Obtient l'ID de la tentative de connexion la plus récente, que cette tentative ait réussi ou non.

getClientInfo

Récupère les informations concernant les propriétés d'informations client prises en charge par le pilote JDBC.

getHoldability

Récupère la fonctionnalité actuelle de mise en attente des objets SQLServerResultSet créés à l'aide de cet objet SQLServerConnection.

getMetaData

Récupère un objet SQLServerDatabaseMetaData qui contient des métadonnées sur la base de données pour lequel cet objet SQLServerConnection représente une connexion.

getTransactionIsolation

Récupère le niveau d'isolation de la transaction actuel pour cet objet SQLServerConnection.

getTypeMap

Récupère l'objet Map associé à cet objet SQLServerConnection.

getWarnings

Récupère le premier avertissement signalé par des appels sur cet objet SQLServerConnection.

isClosed

Indique si cet objet SQLServerConnection a été fermé.

isReadOnly

Indique si cet objet SQLServerConnection est en mode lecture seule.

isValid

Indique si cet objet SQLServerConnection n'a pas été fermé et s'il est toujours valide.

nativeSQL

Convertit l'instruction SQL donnée en grammaire SQL native du serveur de base de données.

prepareCall

Crée un objet SQLServerCallableStatement pour appeler les procédures stockées de base de données.

prepareStatement

Crée un objet SQLServerPreparedStatement pour l'envoi d'instructions SQL paramétrées à la base de données.

releaseSavepoint

Supprime l'objet SQLServerSavepoint spécifié de la transaction actuelle.

rollback

Annule toutes les modifications effectuées dans la transaction actuelle et libère tous les verrous de base de données actuellement maintenus par cet objet SQLServerConnection.

setAutoCommit

Affecte l'état donné au mode de validation automatique pour cet objet SQLServerConnection.

setCatalog

Définit le nom de catalogue spécifié pour sélectionner un sous-espace de la base de données de l'objet SQLServerConnection qui servira d'emplacement de travail.

setClientInfo

Définit la valeur des propriétés d'informations client.

setHoldability

Modifie la fonctionnalité actuelle de mise en attente des objets SQLServerResultSet créés à l'aide de cet objet SQLServerSavepoint pour passer à la fonctionnalité de mise en attente donnée.

setReadOnly

Place cet objet SQLServerConnection en mode lecture seule comme conseil pour le pilote JDBC pour activer les optimisations de base de données.

setSavepoint

Crée un point de sauvegarde sans nom dans la transaction actuelle et retourne le nouvel objet SQLServerSavepoint qui le représente.

setTransactionIsolation

Essaie de modifier le niveau d'isolation de la transaction actuel pour cet objet SQLServerConnection pour passer au niveau donné.

setTypeMap

Installe l'objet TypeMap donné comme mappage de type pour cet objet SQLServerConnection.
