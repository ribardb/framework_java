Méthodes

Nom

Description

addBatch

Ajoute la commande SQL donnée à la liste actuelle de commandes pour cet objet SQLServerStatement.

cancel

Annule l'instruction SQL en cours d'exécution par cet objet SQLServerStatement.

clearBatch

Vide la liste actuelle de commandes SQL pour cet objet SQLServerStatement.

clearWarnings

Efface tous les avertissements signalés sur cet objet SQLServerStatement.

close

Libère immédiatement les ressources de base de données et JDBC pour cet objet SQLServerStatement, au lieu de patienter jusqu'à leur libération automatique.

execute

Exécute l'instruction SQL donnée, qui peut retourner plusieurs résultats.

executeBatch

Envoie un lot de commandes à exécuter à la base de données. En cas d'exécution correcte de toutes les commandes, un tableau des nombres de mises à jour est retourné.

executeQuery

Exécute l'instruction SQL donnée et retourne un seul objet SQLServerResultSet.

executeUpdate

Exécute l'instruction SQL fournie, qui peut être une instruction INSERT, UPDATE, MERGE ou DELETE ; ou une instruction SQL qui ne retourne rien, telle qu'une instruction SQL DDL.

getConnection

Récupère l'objet SQLServerConnection qui a produit cet objet SQLServerStatement.

getFetchDirection

Récupère la direction d'extraction des lignes depuis les tables de base de données. Il s'agit de la méthode par défaut pour les jeux de résultats générés à partir de cet objet SQLServerStatement.

getFetchSize

Récupère le nombre de lignes du jeu de résultats qui représente la taille d'extraction par défaut pour les objets de jeu de résultats générés à partir de cet objet SQLServerStatement.

getGeneratedKeys

Récupère toute clé générée automatiquement qui a été créée suite à l'exécution de cet objet SQLServerStatement.

getMaxFieldSize

Récupère le nombre maximal d'octets pouvant être retournés pour des valeurs de colonne de caractères ou binaire dans un objet SQLServerResultSet, produit par cet objet SQLServerStatement.

getMaxRows

Récupère le nombre maximal de lignes qu'un objet SQLServerResultSet, produit par cet objet SQLServerStatement peut contenir.

getMoreResults

Passe au résultat suivant de cet objet SQLServerStatement.

getQueryTimeout

Récupère le nombre de secondes que le Pilote JDBC Microsoft pour SQL Server doit attendre pour que cet objet SQLServerStatement s'exécute.

getResponseBuffering

Récupère le mode de mise en mémoire tampon des réponses pour cet objet SQLServerStatement.

getResultSet

Récupère le résultat actuel en tant qu'objet SQLServerResultSet.

getResultSetConcurrency

Récupère la concurrence du jeu de résultats pour les objets SQLServerResultSet générés par cet objet SQLServerStatement.

getResultSetHoldability

Récupère la fonctionnalité de mise en attente du jeu de résultats pour les objets SQLServerResultSet générés par cet objet SQLServerStatement.

getResultSetType

Récupère le type de jeu de résultats pour les objets SQLServerResultSet générés par cet objet SQLServerStatement.

getUpdateCount

Récupère le résultat actuel en tant que nombre de mises à jour.

getWarnings

Récupère le premier avertissement signalé par des appels sur cet objet SQLServerStatement.

isClosed

Indique si cet objet SQLServerStatement a été fermé.

isPoolable

Retourne une valeur qui indique s'il est possible d'ajouter une instruction au regroupement d'instructions fourni par l'utilisateur.

isWrapperFor

Indique si cet objet d'instruction est un wrapper pour l'interface spécifiée.

setCursorName

Définit le nom de curseur SQL selon la chaîne donnée, qui est exécutée par les méthodes d'exécution suivantes.

setEscapeProcessing

Définit le mode de traitement d'échappement.

setFetchDirection

Fournit un conseil au pilote JDBC concernant la direction de traitement des lignes du jeu de résultats.

setFetchSize

Fournit un conseil au pilote JDBC concernant le nombre de lignes qui doivent être extraites depuis la base de données, lorsque davantage de lignes sont nécessaires.

setMaxFieldSize

Définit la limite pour le nombre maximal d'octets dans une colonne SQLServerResultSet qui stocke des valeurs de caractères ou binaires selon le nombre d'octets fourni.

setMaxRows

Définit la limite du nombre maximal de lignes que tout objet SQLServerResultSet peut contenir selon le nombre fourni.

setPoolable

Demande qu'une instruction soit regroupée ou non.

setQueryTimeout

Définit le nombre de secondes pendant lequel le pilote doit attendre qu'un objet SQLServerStatement s'exécute, en fonction du nombre de secondes donné.

setResponseBuffering

Définit le mode de mise en mémoire tampon des réponses pour cet objet SQLServerStatement selon String full ou adaptive, sans respect de la casse.

unwrap

Retourne un objet qui implémente l'interface spécifiée, afin d'autoriser l'accès aux méthodes spécifiques au Pilote JDBC Microsoft pour SQL Server.
