Méthode executeUpdate (ISQLServerPreparedStatement)

Exécute l'instruction SQL dans cet objet SQLServerPreparedStatement, qui doit être une instruction SQL INSERT, UPDATE, MERGE ou DELETE ; ou une instruction SQL qui ne retourne rien, telle qu'une instruction DDL.
