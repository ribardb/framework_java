Méthode prepareStatement (ISQLServerConnection)

Crée un objet SQLServerPreparedStatement pour l'envoi d'instructions SQL paramétrées à la base de données.

Liste de surcharge

Nom

Description

prepareStatement (java.lang.String)

Crée un objet SQLServerPreparedStatement pour l'envoi d'instructions SQL paramétrées à la base de données.

prepareStatement (java.lang.String, int)

Crée un objet SQLServerPreparedStatement pour l'envoi d'instructions SQL paramétrées à la base de données ; permet de récupérer des clés générées automatiquement.

prepareStatement (java.lang.String, int[])

Crée un objet SQLServerPreparedStatement pour l'envoi d'instructions SQL paramétrées à la base de données qui est capable de retourner les clés générées automatiquement désignées par le tableau donné.

prepareStatement (java.lang.String, int, int)

Crée un objet SQLServerPreparedStatement qui génère des objets SQLServerResultSet avec le type et la concurrence donnés.

prepareStatement (java.lang.String, int, int, int)

Crée un objet SQLServerPreparedStatement qui génère des objets SQLServerResultSet avec le type, la concurrence et la fonctionnalité de mise en attente donnés.

prepareStatement (java.lang.String, java.lang.String[])

Crée un objet SQLServerPreparedStatement pour l'envoi d'instructions SQL paramétrées à la base de données.
