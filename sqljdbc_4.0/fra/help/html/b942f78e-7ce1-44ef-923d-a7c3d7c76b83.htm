setAuthenticationScheme (SQLServerDataSource)

Indique le genre de sécurité intégrée que votre application doit utiliser.

Syntaxe

public void setAuthenticationScheme(String authenticationScheme);

Paramètres

authenticationScheme

Les valeurs sont "JavaKerberos" et "NativeAuthentication" (valeur par défaut). Pour plus d'informations, consultez Utilisation de l'authentification intégrée Kerberos pour se connecter à SQL Server.
