Présentation du pilote JDBC

Le Pilote JDBC Microsoft pour SQL Server est compatible avec JDBC (Java Database Connectivity) 4.0 et permet un accès fiable aux bases de données Microsoft SQL Server. Le pilote JDBC permet d'accéder à un grand nombre de fonctionnalités introduites dans SQL Server 2005 (notamment la mise en miroir de bases de données), ainsi qu'aux types de données xml, de valeur importante et définis par l'utilisateur. Il prend également en charge la nouvelle fonctionnalité d'isolation de transaction d'« instantané ». En outre, le pilote JDBC prend en charge l'utilisation de l'authentification intégrée.

Les rubriques de cette section présentent de manière générale le pilote JDBC, notamment la configuration système requise, son utilisation et les sources d'informations complémentaires disponibles.
