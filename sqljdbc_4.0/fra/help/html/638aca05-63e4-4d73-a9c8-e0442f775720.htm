Méthode getParameterType (SQLServerParameterMetaData)

Récupère le type SQL du paramètre désigné.

Syntaxe

public int getParameterType(int param)

Paramètres

param

int qui indique l'index de paramètre.

Valeur de retour

int qui indique le code de type JDBC comme défini dans java.sql.Types.

Exceptions

SQLServerException

Notes

Cette méthode getParameterType est spécifiée par la méthode getParameterType dans l'interface java.sql.ParameterMetaData.
