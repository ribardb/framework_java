Méthode getPortNumber (SQLServerDataSource)

Retourne le numéro de port actuel utilisé pour communiquer avec SQL Server.

Syntaxe

public int getPortNumber()

Valeur de retour

Valeur int qui contient le numéro de port actuel.

Notes

Le numéro de port est le numéro de port TCP/IP utilisé lors de l'ouverture d'une connexion socket à SQL Server. Si la propriété portNumber n'est pas définie, la méthode getPortNumber retourne la valeur par défaut 1433.

La méthode setPortNumber n'effectue aucune vérification de plage sur la valeur de port transmise. Vous pouvez transmettre des numéros de port qui ne sont pas valides, tels que 99999, sans déclencher d'erreur.
