Méthode absolute (ISQLServerResultSet)

Déplace le curseur à la ligne donnée dans cet objet SQLServerResultSet.

Syntaxe

public boolean absolute(int row)

Paramètres

row

int qui indique le numéro de la ligne vers laquelle se déplacer. Il peut être égal à 0 ou avoir une valeur positive ou négative.

Valeur de retour

true si le curseur est déplacé vers la position donnée. false s'il se trouve avant la première ligne ou après la dernière ligne.

Exceptions

SQLServerException

Notes

Cette méthode absolute est spécifiée par la méthode absolute dans l'interface java.sql.ResultSet.
