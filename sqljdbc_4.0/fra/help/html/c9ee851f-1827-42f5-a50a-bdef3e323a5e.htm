Méthode getBoolean (java.lang.String)

Récupère la valeur du paramètre désigné en tant que valeur boolean en fonction du nom du paramètre.

Syntaxe

public boolean getBoolean(java.lang.String sCol)

Paramètres

sCol

String qui contient le nom du paramètre.

Valeur de retour

Valeur boolean .

Exceptions

SQLServerException

Notes

Cette méthode getBoolean est spécifiée par la méthode getBoolean dans l'interface java.sql.CallableStatement.
