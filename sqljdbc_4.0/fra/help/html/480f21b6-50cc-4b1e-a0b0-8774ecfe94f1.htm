Méthode createStatement ()

Crée un objet SQLServerStatement pour l'envoi d'instructions SQL à la base de données.

Syntaxe

public java.sql.Statement createStatement()

Valeur de retour

Objet Statement.

Exceptions

SQLServerException

Notes

Cette méthode createStatement est spécifiée par la méthode createStatement dans l'interface java.sql.Connection.
