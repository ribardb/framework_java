Méthode setFetchDirection (SQLServerResultSet)

Fournit un conseil concernant la direction de traitement des lignes de cet objet SQLServerResultSet.

Actuellement, cette méthode n'est pas prise en charge par le Pilote JDBC Microsoft pour SQL Server. Si vous utilisez cette méthode, le pilote JDBC mémorise le paramètre, mais actuellement il n'intervient pas dessus.

Syntaxe

public void setFetchDirection(int direction)

Paramètres

direction

int qui indique la direction d'extraction suggérée. Il peut s'agir de l'une des valeurs suivantes :

ResultSet.FETCH_FORWARD

ResultSet.FETCH_REVERSE

ResultSet.FETCH_UNKNOWN

Exceptions

SQLServerException

Notes

Cette méthode setFetchDirection est spécifiée par la méthode setFetchDirection dans l'interface java.sql.ResultSet.

La valeur initiale de cette méthode est déterminée par l'objet SQLServerStatement qui a produit cet objet SQLServerResultSet. La direction d'extraction peut être modifiée à tout moment.

L'utilisation de cette méthode lorsque le type de curseur est avant uniquement est sans effet.
