Méthode setDateTimeOffset (SQLServerCallableStatement)

Cette méthode a été ajoutée dans le pilote JDBC Microsoft SQL Server version 3.0.

Définit la valeur de la colonne spécifiée selon la valeur Classe DateTimeOffset.

Syntaxe

public void setDateTimeOffset(String sCol, microsoft.sql.DateTimeOffset t)

Paramètres

sCol

Nom d'une colonne.

t

Objet Classe DateTimeOffset.

Exceptions

SQLServerException

Notes

Vous pouvez récupérer une valeur Classe DateTimeOffset avec SQLServerCallableStatement.getDateTimeOffset.

setDateTimeOffset prend la position ordinale de la colonne.
