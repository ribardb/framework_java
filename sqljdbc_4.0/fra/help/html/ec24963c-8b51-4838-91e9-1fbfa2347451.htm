Classe SQLServerStatement

Représente l'implémentation de base de la fonctionnalité d'instruction JDBC.

Package : com.microsoft.sqlserver.jdbc

Implémente :ISQLServerStatement

Syntaxe

public class SQLServerStatement

Notes

La classe SQLServerStatement fournit également un certain nombre de méthodes d'implémentation de classe de base pour l'instruction préparée JDBC et les instructions pouvant être appelées. Le rôle de base de la classe SQLServerStatement est d'exécuter des instructions SQL, puis de retourner des nombres de mises à jour et des jeux de résultats dans l'application utilisateur.

Cette classe prend en charge la désencapsulation dans la classe SQLServerStatement, l'interface ISQLServerStatement et l'interface java.sql.Statement. Pour plus d'informations, consultez Wrappers et interfaces.
