Référence de l'API du pilote JDBC

Le Pilote JDBC Microsoft pour SQL Server fournit une API qui peut être utilisée à l'intérieur du code de programmation Java pour une connexion et une interaction avec une base de données Microsoft SQL Server.

Pour obtenir des informations conceptuelles sur l'utilisation du pilote JDBC, consultez Présentation du pilote JDBC.

Interfaces

Nom de l'interface

Description

Interface ISQLServerCallableStatement

Permet de spécifier le nom de la procédure stockée à appeler avec les paramètres d'entrée et de sortie.

Interface ISQLServerConnection

Représente une connexion JDBC à une base de données SQL Server.

Classe SQLServerDataSource

Représente une liste des propriétés spécifiques à la connexion à une base de données SQL Server à l'aide d'un objet ISQLServerConnection.

ISQLServerPreparedStatement

Représente l'implémentation de base de la fonctionnalité d'instruction préparée JDBC.

ISQLServerResultSet

Représente un jeu de résultats JDBC.

ISQLServerStatement

Représente l'implémentation de base de la fonctionnalité d'instruction JDBC.

SQLServerXAResource

Représente une XAResource pour la gestion des transactions distribuées XA.
