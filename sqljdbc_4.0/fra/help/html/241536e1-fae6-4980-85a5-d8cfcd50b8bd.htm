Méthode getClientInfo (SQLServerConnection)

Récupère les informations concernant les propriétés d'informations client prises en charge par le pilote JDBC.

Liste de surcharge

Nom

Description

Méthode getClientInfo ()

Récupère une liste contenant le nom et la valeur actuelle de chaque propriété d'informations client prise en charge par le pilote JDBC.

Méthode getClientInfo (java.lang.String)

Récupère la valeur de la propriété d'informations client spécifiée.
