Méthode updateBytes (SQLServerResultSet)

Met à jour la colonne désignée avec un tableau de valeurs byte.

Liste de surcharge

Nom

Description

updateBytes (int, byte[])

Met à jour la colonne désignée avec un tableau de valeurs byte en fonction de l'index de colonne.

updateBytes (java.lang.String, byte[])

Met à jour la colonne désignée avec un tableau de valeurs byte selon le nom de la colonne.

Notes

Dans la version précédente du Pilote JDBC Microsoft pour SQL Server, vous pouviez utiliser SQLServerResultSet.updateBytes pour convertir des valeurs entre des tableaux d'octets et les types de données SQL Server date, time, datetime2 ou datetimeoffset. À présent, l'utilisation de cette méthode avec ces types de données lève une exception indiquant que la conversion n'est pas prise en charge.
