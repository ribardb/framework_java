Méthode getConnection (java.lang.String, java.lang.String)

Essaie d'établir une connexion avec la source de données représentée par cet objet SQLServerDataSource à l'aide du nom d'utilisateur et du mot de passe spécifiés.

Syntaxe

public java.sql.Connection getConnection(java.lang.String username,
                                         java.lang.String password)

Paramètres

username

String qui contient le nom d'utilisateur.

password

String qui contient le mot de passe.

Valeur de retour

Objet SQLServerConnection.

Exceptions

java.sql.SQLException

Notes

Cette méthode getConnection est spécifiée par la méthode getConnection dans l'interface javax.sql.DataSource.

L'appel de la méthode getConnection avec un nom d'utilisateur ou un mot de passe non Null entraîne le remplacement des propriétés de nom d'utilisateur et de mot de passe définies sur la classe SQLServerDataSource lors de l'initialisation de l'objet SQLServerConnection. Par exemple, si l'appelant a appelé setUser et setPassword sur la source de données, puis appelle getConnection et fournit un nom d'utilisateur ou un mot de passe non Null, le nom d'utilisateur et le mot de passe définis par setUser et setPassword seront remplacés par le nom d'utilisateur et le mot de passe passés à getConnection.

Le nom d'utilisateur et le mot de passe définis au sein de l'URL à l'aide d'un appel à la méthode setURL ne seront pas modifiés dans ce cas.
