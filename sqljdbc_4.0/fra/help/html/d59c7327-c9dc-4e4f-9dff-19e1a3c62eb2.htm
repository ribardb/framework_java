Méthode setBinaryStream (java.lang.String, java.io.InputStream, long)

Définit le paramètre désigné selon le flux d'entrée spécifié, qui disposera du nombre spécifique d'octets.

Syntaxe

public void setBinaryStream(java.lang.String parameterName,
                            java.io.InputStream x,
                            long length)

Paramètres

parameterName

String qui contient le nom du paramètre.

x

Objet InputStream.

length

long qui indique la longueur en nombre d'octets.

Exceptions

SQLServerException

Notes

Cette méthode setBinaryStream est spécifiée par la méthode setBinaryStream dans l'interface java.sql.CallableStatement.

Si la longueur du flux diffère de ce qui est spécifié dans le paramètre length, le pilote JDBC lève une exception lors de la mise à jour ou de l'insertion de la ligne.

Si la longueur du flux est inconnue, le paramètre length peut être défini sur -1 pour indiquer que le pilote doit accepter le flux, quelle que soit sa longueur. Avec sqljdbc4.jar, nous vous recommandons d'utiliser la méthode JDBC 4.0 Méthode setBinaryStream (java.lang.String, java.io.InputStream) lorsque l'application veut mettre à jour la colonne depuis un flux dont la longueur est inconnue.
