Méthode position (SQLServerNClob)

Récupère la position de caractère de l'objet NClob spécifié ou d'une sous-chaîne dans le NClob, en fonction de la position de départ spécifiée.

Liste de surcharge

Nom

Description

Méthode position (java.sql.NClob, long)

Récupère la position de caractère à laquelle l'objet NClob spécifié searchstr s'affiche dans cet objet NClob.

Méthode position (java.lang.String, long) (SQLServerNClob)

Récupère la position de caractère à laquelle la sous-chaîne searchstr spécifiée s'affiche dans la valeur NCLOB représentée par cet objet NClob.
